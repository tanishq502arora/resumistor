# resumake.io server


### Install dependencies
```
npm install
```

### Starting the server (localhost:3001)
```
npm start
```

### Run tests (lint, flow, jest all at once)
```
npm test
```

### Lint code
```
npm run lint
```

### Fix lint issues
```
npm run fix
```

### Format code
```
npm run format
```

### Typecheck code
```
npm run flow
```

### Install library definitions
```
npm run flow-typed install
```

### Jest tests
```
npm run jest
```

### Building a compiled `lib` output folder
```
npm run build
```
